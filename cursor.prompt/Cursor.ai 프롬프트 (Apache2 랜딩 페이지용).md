Apache2에서 정적으로 호스팅할 수 있는 멋진 랜딩 페이지를 만들어줘.
Tailwind CSS, Google Fonts, AOS 애니메이션, Font Awesome을 포함해서,
디자인이 깔끔하고 반응형이며, Hero 섹션 + 기능 카드 + CTA + 푸터 구조로 구성해줘.
폰트는 Inter, 버튼에는 스크롤 애니메이션을 적용해줘.
모든 코드는 하나의 `index.html` 파일에 작성해줘.
'추천 외부 라이브러리 (정적 사이트에서 사용하기 좋은 것들).md' 파일 참고해서 작성해 줘


페이지 구조는 다음과 같아:
1. Hero 섹션: 큰 타이틀("IoEHub"), 부제목("AI와 IoT를 연결하는 새로운 세상"), CTA 버튼("시작하기")
2. 소개 섹션: IoEHub에 대한 짧은 소개 (2~3문장)
   소개자료는 'IoEHub란.md' 파일 참고 해서 작성 해줘 
3. 기능 섹션: 3개의 기능 카드 ("IoT 장치 연결", "AI 분석 자동화", "실시간 데이터 대시보드")
4. CTA 반복: 버튼("무료 체험 시작")
5. 푸터: 간단한 연락처, © 2025 IoEHub
   '기본정보.md'를 참조해서 작성 해줘 

디자인은 깔끔하고 현대적인 스타일, 흰 배경, 블랙 텍스트, 파란 포인트 색상(#3b82f6).  
CSS는 내부 `<style>` 태그로 포함하고, 반응형으로 만들어줘 (모바일에서도 잘 보이게).
